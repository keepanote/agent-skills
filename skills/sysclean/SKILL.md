# 系统资源清理

压缩进程工作集以释放物理内存,并清理磁盘临时文件。适用于内存吃紧、想快速回收占用的场景。

> 说明:本操作压缩的是进程工作集(把页面挤入待机列表/页面文件),释放数字为**瞬时值**,活跃进程会陆续读回;它**不降低 CPU 占用**。真正的内存大户请看脚本末尾 Top 5 并针对性处理。

## 执行

默认模式(压缩工作集 + 清理临时文件,**不动回收站**),在本 skill 目录下运行:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/clean.ps1
```

可选参数:

- `-IncludeRecycleBin` — 一并清空回收站(**不可恢复**,默认关闭)
- `-MinProcessMB 50` — 仅压缩工作集大于该值的进程(默认 50)
- `-TempAgeDays 2` — 仅删除修改时间早于该天数的临时文件(默认 2)

以管理员身份运行可覆盖更多进程与 `Windows\Temp`。

脚本完成后报告关键数字:压缩进程数、挤出工作集、删除临时文件与磁盘释放、清理前后内存对比。
